# Elysia Protocol Index (엘리시아 프로토콜 인덱스)

이 디렉터리는 엘리시아의 비전, 원칙, 아키텍처와 운영 방법을 정의하는 단일 진실원(Single Source of Truth)입니다.

프로토콜의 목적은 다음과 같습니다.
1. 단일 진실원: 모든 참여자가 같은 비전과 목표를 공유하도록 함
2. 지속 가능한 성장: 시간이 지나도 확장/계승 가능한 구조 유지
3. 상호 운용성: 모듈과 문서가 일관된 규약으로 함께 동작하도록 함
