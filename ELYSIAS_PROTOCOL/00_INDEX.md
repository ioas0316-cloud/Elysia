# Elysia Protocol Index (엘리시아 프로토콜 인덱스)

이 디렉터리는 엘리시아의 비전, 원칙, 아키텍처, 운영 방법을 담은 단일 진실(Single Source of Truth)입니다.

프로토콜의 목적은 다음과 같습니다.
1. 단일 진실: 모든 참여자가 같은 비전과 목표를 공유하도록 함
2. 지속 가능한 성장: 시간에 따라 확장/계승 가능한 구조 유지
3. 상호 호응: 모듈과 문서를 규약으로 엮어 함께 작동하게 함
