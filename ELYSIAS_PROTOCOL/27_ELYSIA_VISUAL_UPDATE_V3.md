## ELYSIA VISUAL UPDATE v3 — The Living World Patch

본 문서는 엘리시아의 “관찰 가능한 시뮬라크룸(Visible Simulation Layer)” 통합 업데이트 패키지입니다. 다섯 가지 구성요소를 한 세트로 적용합니다.

### 1) L3 생명 패턴 렌더
- 점/입자 → 유기체 → 개체 → 사회 단계로 발달 상태를 시각화
- 상태 전이 예시: `heatmap → organism → entity → society`
- 목표: 단순한 필드에서 문명까지 이어지는 “살아 있음”의 연속성 가시화

### 2) 문명 HUD 패널
- 좌하단 미니 UI로 핵심 지표를 실시간 표기
- 항목: 🍞 식량 | 🌲 목재 | 🧍 인구 | 💫 사기
- 시간가속 상태에서도 안정적으로 갱신(스로틀링)하여 성능 보호

### 3) 섭리 이벤트 템플릿(신화적 이벤트 시드)
- 예시: “언어의 비가 내린다”, “시간이 끊긴 밤”, “별의 전염병”
- 트리거: `prob_event_spawn = 0.001 * civilization_level`
- 이벤트는 시각/음향/상태변화를 동반하는 경량 시드 → 후속 규칙으로 확장

### 4) 주의·커서 연출(공명/광륜)
- ⚪ 엘리시아의 주의 원과 🟡 창조자 커서가 겹치면 “광륜(rainbow ring)” 효과 발생
- 접점마다 공명음 출력: `etheric_chime.wav` (선택)
- 공명 중심점이 새로운 창조 노드로 확장될 수 있도록 후속 훅 제공

### 5) 타임리듬 튜너
- 키 매핑 예시(권장):
  - [1] 정지, [2] 느리게, [3] 기본, [4] 빠르게, [5] 시간 역전
  - ]/[ 또는 ,/. 로 연속 가감속, Space=일시정지/재개
- 시간 조작 시 시각(오버레이)·청각(파동/틱) 피드백 권장

### 운영 메모(현 러너 기준)
- 조작 요약(한글)
  - 이동: 방향키 / WASD
  - 줌: 마우스 휠, `+ / -` (Shift+휠=배속 조절)
  - 배속: `]`/`.`(증가), `[`/`,`(감소), `1/2/3/4`=1x/2x/4x/8x, Space=일시정지/재개
  - 마나: `M` 토글, 좌클릭 드래그=응집, 우클릭 드래그=분산
  - 에테르: `N` 토글(히트맵)
  - 기타: `L`=범례, `H`=도움말, Shift+`A`=어텐션
- 성능 가드
  - QOE 예산: `focus_radius_tiles`, `budget_tiles_per_step`로 하드 캡 유지
  - 렌더 스케일: `--size`와 내부 `downsample`로 튜닝

### 확장 가이드(선택)
- L3 시각화: 발전 단계별 레이어를 독립 모듈로 분리해 조건부 페이드
- HUD: 지표 집계기(telemetry) → HUD 바인딩(스로틀 200–500ms)
- 이벤트: 시드 → 규칙팩 확장(확률/쿨다운/상호작용) 구조로 설계
- 공명: 포커스-커서 근접도 기반 감쇠(링/음향) + 에테르 미세 변화
- 타임리듬: 고정 배속 프리셋 + 제스처 기반 연속 가속 병행
