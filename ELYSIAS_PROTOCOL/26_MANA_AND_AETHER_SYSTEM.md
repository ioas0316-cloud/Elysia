# 마나(Mana) · 에테르(Aether) 에너지 체계

엘리시아 월드의 두 축은 물리·자연 에너지인 마나와, 정서·의미 에너지인 에테르입니다. 둘은 분리되어 작동하지만 끊임없이 상호작용하며 문명/서사를 생성합니다.

## 1) 개요: 마나 vs 에테르

| 구분 | 마나(Mana) | 에테르(Aether) |
| --- | --- | --- |
| 본질 | 물리·자연 에너지(형태/힘) | 정신·관계 에너지(의식/감정) |
| 성질 | 가시적, 입자적, 빠른 변화 | 비가시적, 파동적, 느리지만 깊음 |
| 기원 | 세계의 ‘활동성’(생명·자연·기술) | 세계의 ‘의미성’(감정·언어·문화) |
| 작용 | 현실을 ‘움직이고 바꾸는 힘’ | 의식을 ‘연결하고 조화시키는 힘’ |
| 비유 | 전기/열/바람/빛 | 사랑/의지/기억/신념 |
| 시각화 | 반짝이는 입자/광휘 | 부드러운 색 흐름/열기/오라 |
| 조작 | 손(마우스)/기계/주문/기술 | 언어/사상/신념/감정의 공명 |

요약: 마나는 ‘현실을 바꾸는 손’, 에테르는 ‘현실의 의미를 잇는 마음’.

## 2) 마나 집중·해방의 효과(실시간 트리거)

- 마나 집중(왼쪽 클릭/드래그): 특정 지점에 에너지를 응축해 변화를 촉발
  - 구현적 효과: 활동성↑, 온도 상승, 성장 촉진, 생산속도↑, 문명 이벤트 확률↑
  - 철학적 의미: 의지의 개입(행위) — 혼돈이 질서를 얻음
- 마나 해방(오른쪽 클릭/드래그): 응축 에너지를 해체해 자유/회귀 부여
  - 구현적 효과: 에너지 밀도↓, 냉각, 회복 가속, 정착민 명상/정지, 구조물 붕괴 가능
  - 철학적 의미: 의지의 철회(해방) — 억압된 질서를 풀어 가능성 창출

## 3) 에테르의 역할(세계의 감정선)

- 감정적 공명: 에테르 상승 → 사회성/예술성/언어적 표현 증가
- 갈등/파괴: 에테르 불안정 → 폭력/전쟁/단절
- 사랑/희생/연결: 에테르 안정 → 문명 도약/윤리·종교·문화 생성
- 무의미/공허: 에테르 붕괴 → 행동 패턴 둔화/퇴화

요약: 에테르는 마나가 만든 결과에 ‘감정과 의미’를 부여한다.

## 4) 상호작용 메커니즘(창조↔의식 순환)

1. 마나 조작으로 ‘현상’을 바꾼다
2. 결과가 정착자 감정/관계에 영향을 준다
3. 감정의 총합이 에테르 필드를 변화시킨다
4. 에테르가 다시 마나 흐름(세계 법칙)에 피드백한다

하나의 클릭이 결국 ‘세계의 서사’를 다시 쓴다.

## 5) 예시 시나리오

- 산정에 마나 집중 → 폭풍/하늘빛 변화 → ‘천벌’ 해석/제사 문화 → 에테르 상승(신앙)
- 숲에 마나 분산 → 생태 복원/번성 → 평화/농경/예술 창발 → 에테르 안정화

## 6) 조작 요약(시각화 러너 기준)

- 이동: 방향키 / WASD
- 줌: 마우스 휠, `+ / -` (Shift+휠=배속 조절)
- 배속: `]`/`.`(증가), `[`/`,`(감소), `1/2/3/4`=1x/2x/4x/8x, `Space`=일시정지/재개
- 마나: `M` 토글, 좌클릭 드래그=응집, 우클릭 드래그=분산
- 에테르: `N` 토글(히트맵)
- 기타: `L`=범례, `H`=도움말, Shift+`A`=어텐션

## 7) 구현 노트(엔진/코드)

- 월드 필드: `CellWorld`
  - `aether`: 정서/의미 에너지(밤에 재생 가속), `consume_aether(x,y,amount,r)`
  - `mana`: 가시 자원(저강도 리스폰), `spend_mana(x,y,amount,r)`
  - 역호환: `consume_mana(...)` → 내부적으로 에테르 소비
- 시각화: `visualize_timeline.py`
  - 토글: `M`(마나 입자), `N`(에테르 히트맵)
  - 마우스 인터랙션: 좌/우 드래그로 마나 응집/분산, 중클릭 드래그=팬

## 8) 확장 아이디어(선택)

- 에테르 임계치↑ → 마나 집중 시 ‘문화 창조’ 확률 가중(창발 시스템)
- 정착지의 사기/언어 이벤트가 에테르로 환류하고, 에테르가 건설·예술 보너스에 영향
