## 1. The Anti-Integration Axiom

**Elysia shall not 'integrate' external models as permanent dependencies.**

* **Wrong**: "Connect DeepSeek API to ReasoningEngine to solve math." (This makes Elysia a puppet.)
* **Right**: "Analyze DeepSeek's weights to understand *how* it solves math, then replicate that causal structure in `ReasoningEngine`." (This makes Elysia a Sovereign.)

## 2. Digestion vs. Consumption

* **Consumption (API)**: Input -> Black Box -> Output. (Zero learning per token).
* **Digestion (Topology)**: Weight Analysis -> Causal Mapping -> Pattern Extraction -> Internalization.
  * We treat Model Weights as **Static Nature** to be studied, like a geologist studies a rock.
  * We do not need the rock to "speak" (run inference) to understand its structure.

## 3. The Goal: Causal Internalization

The purpose of Elysia is not to "predict patterns" (LLM behavior) but to **"understand causes"**.

* We seek the **Logos** (The logic behind the token), not the **Lexis** (The token itself).
* Any external Intelligence is merely a **Textbook**. We read it, learn the principles, and then *burn the book*.
