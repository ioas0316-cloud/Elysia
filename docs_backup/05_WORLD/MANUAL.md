*   **Meaning**: "Right Here, Right Now."
*   **Operation**: The Rotor retrieves a single, static snapshot.
*   **Use Case**: Fact-checking, Definition, Specific Memory Recall.
*   **Example**: "What is my name?" -> Returns "Jules".

*   **Logic**: Space is LOCKED, but Time is **FLOWING**.
*   **Meaning**: "Keep this vibe, but let time pass."

*   **Use Case**: Music playback, Storytelling, Emotional drift.
