# ⚖️ 사법부 가혹 테스트 시나리오 (Tension Divergence Tests)

본 디렉토리는 사법부 감시 데몬이 행정부와 입법부를 탄핵하기 위해 사용하는 **가혹 스트레스 테스트**의 저장소입니다.

## 핵심 수사 원칙
1.  **하드코딩 적발**: 정적 분석을 통해 C/CUDA 커널 내의 꼼수를 적발합니다.
2.  **면 매핑 오차율 계측**: 유속 입력 시 O(1) 평탄화가 무너지거나 지연 시간이 증가하면 즉각 반려합니다.
3.  **대시보드 기만 추적**: 쥴스의 대시보드가 실제 레지스터를 읽지 않고 캐싱된 가짜 쇼를 하는지 시간차/메모리 트래킹을 통해 검증합니다.

## 테스트 파일
*   `test_tension_divergence.py`: (예정) 텐션 분기 시 오차율을 극한으로 테스트.
*   `test_mocking_detection.py`: (예정) 대시보드의 메모리 참조를 후킹하여 가짜 연산 적발.
