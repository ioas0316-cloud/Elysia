1. **적절한 위치 선택**
   - Root: 핵심 필수 문서
   - docs/: 상세 가이드
   - Protocols/: 설계 문서
   - docs/Manuals/: 매뉴얼

   ```markdown
   # 문서 제목
   
   > **목적**: 명확한 목적 설명
   
   **버전**: 7.0
   **최종 업데이트**: YYYY-MM-DD
   
   ---
   
   ## 목차
   ...
   ```

3. **상호 참조 추가**
   - 관련 문서 링크
   - "다음 단계" 섹션

   - 새 문서 추가 시 이 파일도 업데이트
