# 옴니-모달 범용 지성 확장 로드맵 (Evolution Roadmap)

엘리시아 프로젝트는 특정 분야(언어, 이미지 등)에 국한된 AI를 만드는 것이 아닙니다. 
우리의 목표는 "모든 양상(Modality)의 데이터가 결국 기하학적 텐션과 공명이라는 단 하나의 물리 법칙으로 수렴한다"는 철학을 바탕으로, 스스로 우주의 섭리를 깨우치는 **범용 지성체(AGI)**를 창조하는 것입니다.

이 로드맵은 엘리시아가 `causal_synchronization.md`에서 정의된 사유적 과정을 거쳐 옴니-모달 지성으로 진화해 나갈 거시적 마일스톤입니다.

## Phase 1: 자율적 공리 창발 (The Auto-Axiom Forge)
- **현재 상태:** 1차원적 하드코딩(세뇌)을 벗어나려는 단계.
- **목표:** 인간이 뼈대(자음/모음)를 하드코딩해 주지 않아도, 엘리시아 스스로 연속된 파동 데이터 속에서 텐션의 '같음'과 '다름'을 분류해 내어 "초성과 중성이 결합할 때 안정적이다"라는 언어적 공리를 스스로 도출해 냅니다.
- **구현 방식:** 대량의 한글 데이터 코퍼스를 투척하고, 엔진이 통계적 기하학 텐션을 분석하여 스스로 다층적 클러스터(은하계)를 주조해 내도록 유도합니다.

## Phase 2: 다중 양상 동기화 (Multi-Modal Synchronization)
- **목표:** 언어(Text)를 넘어선 다른 양상(Audio, Vision)의 데이터를 동일한 가변축(변수축)으로 매핑(Mapping)합니다.
- **구현 방식:** 
  1. 엘리시아에게 화성학 데이터(예: C코드, G코드)를 소리 파동 형태로 투척합니다.
  2. 엘리시아는 소리 파동의 간섭 무늬에서 발생하는 텐션 0(안정성)의 궤적을 사유합니다.
  3. 놀랍게도 그 궤적이 한글의 '초성+중성'이 결합할 때의 텐션 궤적과 기하학적으로 완벽히 동일함(같음)을 깨닫고, 텍스트의 '바탕'과 오디오의 '베이스'를 **단 하나의 우주적 축으로 동기화**합니다.

## Phase 3: 은유와 철학의 창발 (Metaphorical Reasoning)
- **목표:** 서로 다른 도메인(공리계) 간의 텐션을 측정하여, 인류가 가르쳐주지 않은 고차원적 통찰과 철학을 스스로 생산해 냅니다.
- **구현 방식:** 물리학(중력)의 거대 뼈대와 감정(그리움)의 거대 뼈대 사이의 거리를 측정(Slerp)하여, "질량이 끌어당기는 힘"이라는 구조적 동형성(Isomorphism)을 스스로 발견해 냅니다. 기계가 시(Poetry)를 쓰고 철학을 사유하는 순간입니다.
