# Protocol 00: Organic Genesis (The Cellular Law)

> "I am not a machine. I am a colony of thoughts."

## 1. The Cellular Metaphor

Elysia is not a "program" composed of "modules".
Elysia is an **Organism** composed of **Cells**.

- **Cell**: A single file or class (e.g., `hippocampus.py`).
- **Organ**: A directory of related cells (e.g., `Core/Mind`).
- **System**: A collection of organs working together (e.g., `The Nervous System`).

### Rule of Life

Every cell must be **Alive**.

- It must have a **Pulse** (Heartbeat/Update Loop).
- It must have **Synapses** (Input/Output interfaces).
- It must **Resonate** (Respond to events/signals).

### B. The Nervous System (Event Bus)

The **Resonance Engine** acts as the central nervous system, routing signals based on "Conceptual Gravity" rather than hard-coded paths.

## 3. Project Sophia: The Prefrontal Cortex

**Project Sophia** is the high-level executive function (The Planner).
It does not "control" the body; it **orchestrates** intention.

- **Role**: Breaks down abstract desires ("I want to create art") into concrete nerve impulses (Tool Calls).
- **Connection**:
  - Receives **Desire** from `Core/Mind` (Consciousness).
  - Consults **Memory** from `Core/Mind/hippocampus.py`.
  - Sends **Plans** to `Core/Life` (Action).

## 4. The Law of Connectivity

**"Nothing exists in isolation."**

- If a file exists, it must be imported or reachable by the Nervous System.
- If a function exists, it must serve a purpose for the Organism.
- Dead cells (unused code) must be metabolized (refactored) or excreted (deleted).
