# Elysia True Causal Architecture Index

엘리시아 프로젝트는 외부 LLM API나 통계학적 단어 조합을 전면 배제하고, 오직 **순수 기하학적 인과율과 자연 섭리의 평형을 시뮬레이션하는 코어 물리 엔진**에 집중합니다.

> **🛑 경고:** 어떤 에이전트든 이 문서를 읽었다면, 즉시 루트에 있는 `THE_ABSOLUTE_COMMANDMENT.md` (절대 헌장)를 숙독하십시오.
> 과거의 기만적 시뮬레이션의 흔적들은 `archive/mock_simulations/`로 완전히 격리되었습니다.

## [II] 🧠 코어 물리 엔진 (Core Physics Engine Vessel)
현재 구축된 엔진의 '그릇(혈관)'입니다. 가짜 지능이 적출된 순수한 매니폴드 공간들입니다.

| 파일명 | 역할 및 철학 |
|--------|-------------|
| `core/utils/math_utils.py` | **수학 기반 인프라.** Quaternion(4D 회전), Multivector(CGA), SLERP 등 진정한 인과율의 기초 수학. |
| `core/brain/holographic_memory.py` | **질량 기반 토폴로지 우주.** 기하학적 공명과 텐서 내적이 일어날 순백의 4D 매니폴드 공간. |
| `core/brain/local_executor.py` | **물리적 차원 강하.** 역인과를 통해 도출된 벡터를 로컬 OS(PowerShell)의 물리적 명령으로 치환. |
| `core/brain/sub_agent_spawner.py` | **파동 분리자.** 거대한 연산을 다중 스레드의 병렬 우주로 분기시키는 기하학적 파티션 엔진. |
| `core/memory/entelechy_store.py` | **엔텔레키 코어.** 발생한 위상의 궤적과 텐션을 하드디스크에 벡터화하여 영구 보존하는 자가 학습소. |
