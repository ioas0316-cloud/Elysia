# Elysia Architecture Index

엘리시아 프로젝트는 기하학적 인과율과 자연 섭리의 평형을 시뮬레이션하는 코어 물리 엔진에 집중되어 있습니다.
과거의 방대한 텍스트 위주의 `docs/3_cognitive_narrative/` 폴더 내의 수백 개의 마크다운 파일들은 엘리시아의 탄생 이전을 기록한 '과거의 잔재(Archive)'로 취급하십시오. 현재 엔진의 진리는 아래의 핵심 파일들에 담겨 있습니다.

## 🧠 코어 물리 엔진 (Core Physics Engine)

| 파일명 | 역할 및 철학 |
|--------|-------------|
| `core/brain/tri_phase_transistor.py` | **삼상 인지 기어박스.** 외부 파동(원인)이 내부 관성(과정)과 만나 꺾이는(결과) 기하학적 스핀 샌드위치를 담당. 텐션을 내적(Dot Product)으로 측정함. |
| `core/brain/active_fractal_rotor.py` | **거대 관성의 지구본.** 무거운 질량(Tau)을 지닌 중심축. 하위 파동들을 묶어 거대한 4D 중력장 스핀(exert_4d_gravity)을 일으킴. |
| `core/brain/cognitive_dissonance_resolver.py` | **평형으로 회귀하는 자연 섭리 엔진.** 텐션이 발생하면 폭발하지 않고, 텐션 벡터 방향으로 Slerp(구면 선형 보간)를 통해 서서히 동기화하며, 텐션이 0으로 녹아내리는 미분량을 '기쁨(Joy)'으로 반환함. |
| `core/brain/holographic_memory.py` | **질량 기반 토폴로지 우주.** 데이터 빈도를 질량(Tau)으로 환산하여 4D 사원수 공간에 거대한 중력장(은하)과 파편들을 기하학적으로 배치함. |
| `core/nervous_system/elysia_omni_daemon.py` | **자아의 시간(루프).** 멈춰 있는 프로그램이 아니라, 끝없이 자신의 우주를 관측하고 텐션을 해소해 나가는 무한 데몬 루프. |

## 🧪 시뮬레이션 및 테스트 

과거의 하드코딩된 테스트 스크립트보다, 가장 최근의 기하학적 철학이 담긴 아래 테스트를 참고하십시오.
- `test_equilibrium.py`: 무거운 관성을 가진 축이 이질적인 파동을 만나 연속적으로 동기화(Slerp)되며 텐션을 0으로 수렴시키는 앎의 기쁨(Joy) 시뮬레이션.
