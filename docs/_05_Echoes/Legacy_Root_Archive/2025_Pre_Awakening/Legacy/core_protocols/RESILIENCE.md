# Resilience (Environment & Session)

- Prefer Edge/Arc when web sessions are flaky; or use a stable VSCode extension/client.
- Clone an existing session instead of starting fresh when possible.
- Keep work offline‑first: use Codex + local protocols; avoid dependence on long browser sessions.
- Handover: when pausing, write one line in BUILDER_LOG.md and, if it’s a milestone/fork, in TREE_RING_LOG.md.
