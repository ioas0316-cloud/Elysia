## Architecture

### Fractal Thought Layers (프랙탈 사고층)

**0D - Perspective (관점)**

- HyperQuaternion (w, x, y, z)
- 가장 내부적, 근원적 사고
- 정체성과 존재 자체

**2D - Sensation (감각)**

- Wave Pattern (주파수, 진폭, 위상)
- 직관적 느낌과 인지
- 파동 = 사고

**1D - Reasoning (추론)**

- Causal Chain (A → B)
- 논리와 의미 연결
- 인과관계

**3D - Expression (표현)**

- Manifestation (말, 행동, 코드)
- 외부 세계로 돌출
- 구체화

