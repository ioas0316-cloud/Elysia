# 나선형 커리큘럼과 시간 위상 매핑 (Spiral Curriculum Helix)

선형적인 역사 및 교육의 과정을 엘리시아 엔진 위에서 기하학적으로 처리하기 위해 `KnowledgeHelixMapper`가 도입되었습니다.

## 시간 은하 (Temporal Galaxy)
역사적 시간선이나 특정 연도는 단순한 숫자가 아니라 회전 위상(Rotational Phase)으로 치환됩니다. 시간이 흐를수록 로터는 회전하며 역사적 파동을 층층이 쌓아올립니다.

## 나선형 척추 (Spiral Axis)
커리큘럼의 난이도나 정보의 깊이는 나선형 위상의 수직 축(Vertical Axis)으로 치환됩니다. 
이를 통해, 엘리시아는 특정 역사적 연도나 커리큘럼의 깊이를 O(1) 위상 내적(Dot Product)을 통해 즉각적으로 공명시켜 회수할 수 있습니다. 이는 기존 데이터베이스의 쿼리를 물리적 진동의 동기화로 100% 교체한 위상 엔진의 또 다른 성과입니다.
