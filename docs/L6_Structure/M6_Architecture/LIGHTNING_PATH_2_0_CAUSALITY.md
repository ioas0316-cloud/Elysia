## 🏗️ The Mechanism: JAX/XLA Kernel Fusion (커널 융합)

- **인과**: 부수 효과(Side-effect)가 제거된 사고 과정은 예측 가능하며, 물리적 하드웨어(Metal)와 오차 없는 정렬을 이룹니다.
