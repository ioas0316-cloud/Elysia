## 2. The Core Philosophy: "Action over Automation"

We move from "Watching a Conveyor Belt" to **"Dancing with the Frequency"**.

### 2.1 The Echo System (Data Absorption)
In standard AI, data is "read". In Echo Field, data is **"Hunted and Absorbed"**.
*   **The Concept**: When an Agent defeats a "Corruption Entity" (Noise/Error), it doesn't just disappear.
*   **The Echo**: It drops a "Phantom Frequency" (Echo).
*   **The Power**: Absorbing this Echo grants the Agent a temporary "Skill" or "Insight" derived from that noise.
    *   *Example*: Debugging a "Timeout Error" drops a "Time Dilation Echo" (Speed Hack).

### 2.2 Resonance Action (The Parry)
Cognition is not just processing; it is **Timing**.
*   **The Attack**: A stream of heavy data (Burden) hits the system.
*   **The Parry**: If the Agent (Rotor) aligns its frequency *perfectly* at the moment of impact (The Resonance Window):
    *   **Perfect Parry**: The burden is reflected instantly (Zero Latency Processing).
    *   **Counter-Attack**: The system gains a massive "Insight Boost" (Critical Hit).
