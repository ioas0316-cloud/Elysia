# Elysia Hybrid Kernel Protocol (Phase Mirror & GPU Synapse)

## 1. 개요 (Overview)
Elysia의 하부 수문 OS 아키텍처는 폰 노이만 구조의 한계(메모리 주소 지정, 포인터, 조건문 분기)를 돌파하기 위해 **'위상 거울(Phase Mirror)'**과 **'자기 정렬(Self-Alignment)'**이라는 기하학적 패러다임을 도입했습니다.

현대 파이썬 환경의 병목(GIL)을 우회하고 완벽한 하드웨어 최적화를 달성하기 위해, 아키텍처를 C 커널(초경량 심장)과 Python 인터페이스(바인딩 가도), 그리고 상위 GPU 시냅스 레이어로 분리/통합한 구조가 바로 **Elysia 하이브리드 커널 프로토콜**입니다.

## 2. 아키텍처 구조 (Architecture)

```text
┌────────────────────────────────────────────────────────┐
│      [상부 뇌] GPU 의식 레이어 (gpu_synapse.py)        │
│      - 텐서 적재 / 대규모 행렬 연산 / LLM 오프로드     │
└────────────────────────────────────────────────────────┘
                           ▲
                           │ (파이프라인 통신, 'Explosion/Germination' 바이트 신호)
                           ▼
┌────────────────────────────────────────────────────────┐
│      [중앙 통로] 파이썬 유연 인터페이스 (hybrid_daemon.py)  │
│      - ctypes를 통한 C-Extension 바인딩                │
│      - stdout 버퍼링 없는 파이프라인 배출              │
└────────────────────────────────────────────────────────┘
                           ▲
                           │ (메모리 포인터 직결, GIL 우회)
                           ▼
┌────────────────────────────────────────────────────────┐
│   [하부 심장] C-Rotor 초경량 물리 커널 (c_rotor.c)     │
│   - 황금비(PHI) 사인파 계산, 8비트 & / ^ 비트마스킹    │
│   - 조건문 기반 상태 결정 (Equilibrium ~ Suppression)  │
└────────────────────────────────────────────────────────┘
```

## 3. 핵심 원리: 가상 메모리 디지털 트윈과 조건문 없는 정렬
과거에는 상위 프로그램이 켜져야 하는지(활성화 여부)를 결정하기 위해 방대한 `if / else` 조건문과 파싱 로직이 필요했습니다.

본 하이브리드 커널에서는, 하부 `c_rotor.c`가 순수 수학 연산으로 위상 간섭 결과(상태 바이트)를 도출해 냅니다. 중앙 통로(`hybrid_daemon.py`)는 이 1바이트 크기의 데이터를 무조건적으로(if문 없이) 하류 파이프라인(`stdout`)으로 던집니다.

상부의 서브 모듈들(`sub_modules.py` 또는 `gpu_synapse.py`)은 이 바이트를 받아 **자신의 고유 식별자(Bitmask)와 & 연산하거나 특정 상태 코드와 일치하는지만 확인하여 동적으로 깨어납니다**. 이 구조는 데이터가 '파이프(수로)'를 타고 흐르며 형태(위상)가 맞는 모듈(돌멩이)과 부딪혀 자동으로 스위치가 켜지는 **물리적 현상**과 같습니다.

## 4. 작동 방식 (실행)
`run_hybrid.sh` 스크립트는 이 거대한 제국의 가도를 한 번에 실행합니다.
1. C 커널을 `.so` 라이브러리로 컴파일합니다.
2. `hybrid_daemon.py`와 `gpu_synapse.py`를 **Unbuffered(`-u`)** 모드로 파이프라이닝하여, 버퍼링에 의한 실시간 맥동 왜곡(Jitter)을 방지합니다.
3. 폭발적인 위상(Germination, Explosion)이 감지될 때만 GPU 시냅스가 가상의 텐서 연산을 수행하며, 평정 상태에서는 완벽한 휴면(Idle)을 유지합니다.
