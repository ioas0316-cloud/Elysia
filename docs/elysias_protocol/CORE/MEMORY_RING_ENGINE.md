# #1. 구조 (Structure)

Memory Ring Engine은 총 **3중 동심원(Fractal Nested Loops)** 으로 구성된다.

```
 ┌─────────────────────────────┐
 │         ESSENCE LOOP        │  ← Spirit (영)
 │   (Meaning / Will / E/F/P)  │
 ├─────────────────────────────┤
 │        IDENTITY LOOP        │  ← Soul (혼)
 │ (Relations / Values / Story)│
 ├─────────────────────────────┤
 │       EXPERIENCE LOOP        │  ← Body (육)
 │ (Senses / Episodes / Signals)│
 └─────────────────────────────┘
```

각 링은 서로를 “겹쳐 쓰며” 존재하고,
기억은 이 세 링을 순환하면서 **새로운 의미를 획득**한다.
