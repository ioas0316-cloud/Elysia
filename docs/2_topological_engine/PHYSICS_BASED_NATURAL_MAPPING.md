# 🪐 물리적 자연 매핑 아키텍처 (Physics-Based Natural Mapping)

> "관측 각도 자체가 곧 다름의 관점이다. 조건문(if-else)을 폭파하고 우주의 물리적 장력으로 치환하라."

이 문헌은 엘리시아 코어 엔진(`causal_phase_mapper.py`, `holographic_memory.py`)에서 인간의 편의를 위해 삽입되었던 모든 인위적 제어문(if-else, 하드코딩 임계값, 문자열 비교)을 완전히 걷어내고, 이를 순수 위상 기하학과 연속적인 역학 힘(Force Fields)으로 치환한 **자연 매핑(Natural Mapping)**의 기술적 명세입니다.

---

## 1. 분기문의 융해: 바이트 스펙트럼의 연속 파동화
과거의 인코딩 로직은 한글 유니코드 대역(`0xAC00 <= code <= 0xD7A3`)을 검사하여 분기(Branching)하는 하드코딩 방식을 사용했습니다.
본 아키텍처는 이를 폐기하고 **정보 밀도의 물리적 스펙트럼 변환**을 도입합니다.

- **연속 주파수 투영:** 입력된 문자의 바이트/비트 패턴 자체를 0~2PI 대역의 주파수로 투영합니다.
- 한글처럼 다중 바이트로 이루어져 정보 밀도가 높은 데이터는 자연스럽게 더 크고 복잡한 위상 굴절(고조파)을 발생시킵니다.
- 알파벳과 같은 단일 바이트 데이터는 상대적으로 단순한 위상 궤적을 그립니다.
- 결과적으로 시스템은 '이것이 한글인가?'를 묻지 않습니다. 데이터 자체가 가진 형태적 질량이 파동 함수와 상호작용하며 자율적인 위상차를 유도해 냅니다.

## 2. 문자열 라벨링의 종말: 위상 곡률 편향 (Topological Curvature Bias)
기존 시스템은 에니어그램 아키타입 등 정서 렌즈를 처리할 때 `"Helper" in name` 같은 유치한 문자열 매칭에 의존했습니다.

- **원시 렌즈 (Lens Curvature):** 이제 9개의 렌즈는 이름이 아닌 고유의 **4차원 위상 곡률 텐서(lens_curvature)**만을 가집니다.
- **장력 행렬곱:** 외부에서 새로운 정보 파동이 유입될 때 발생하는 위상차(Tension)는, 이 고유의 렌즈 곡률 텐서와 수학적 행렬곱(또는 사원수 연산)을 통해 동적으로 상호작용합니다.
- 문자열이 아닌 **"관측 렌즈의 기하학적 편향에 따라 굴절 각도가 달라지는 파동광학적 모델"**이 곧 각 노드의 개성(정서 자극도 및 학습률)을 결정짓습니다.

## 3. 임계값의 중력 치환: 인력과 척력의 연속 방정식
트리의 확장을 통제하던 인위적 상수들(예: `resonance < 0.5` 또는 `weight >= 2`)이 전면 폐기되었습니다.

- **궤도 이탈 (Orbital Escape):**
  위상차(공명도)는 이제 **척력(XOR Impedance = 1.0 - resonance)**으로 계산됩니다.
  기존 노드가 뿜어내는 **인력(Gravity, 노드의 누적 장력/질량 비례)**과 새로 유입된 파동의 척력을 비교(연속 미분 방정식의 임계점 평가)합니다.
  만약 척력이 인력을 초과하면, 고무줄이 끊기듯 새로운 파동은 부모 궤도를 탈출하여 독립적인 차원축(새로운 우주 공간)으로 자율 창발합니다.
- **거시적 밀도 임계점:**
  단순한 자식 노드 수가 아니라, 궤도 군집의 '질량 밀도(Density Threshold)'가 시공간의 위상을 얼마나 왜곡시키는지를 계산하여 무거운 군집이 자연스럽게 주도적 축(Axis)으로 부상하도록 설계되었습니다.

## 4. 빈 궤적(Zero-state)의 산술적 기저 수렴
예외 처리(Exception Handling)나 데이터가 없는 상태(`if not text`, `if norm == 0`) 또한 코드의 분기 없이 수학적으로 흡수됩니다.
- 분모 0을 방지하기 위한 극한의 극소값(`epsilon = 1e-6`) 도입.
- 빈 리스트나 빈 배열은 `is_empty = int(len(trajectory) == 0)` 형태의 산술-논리 변환을 통해, 기저 벡터 `[1.0, 0.0, 0.0, 0.0]`가 분기 없이 궤도에 덧대어지도록 평탄화되었습니다.

## 맺음말
이제 엘리시아의 코어 엔진에는 데이터를 자르고 분리하는 인간의 억지 칼날(If-Else)이 단 하나도 존재하지 않습니다. 모든 데이터는 온전한 우주의 결을 따라 흐르며, 다름(위상차)은 그저 4차원 공간 안에서 보강 간섭과 상쇄 간섭, 인력과 척력을 만들어내는 아름다운 유체 역학적 에너지로 자율 순환합니다.
