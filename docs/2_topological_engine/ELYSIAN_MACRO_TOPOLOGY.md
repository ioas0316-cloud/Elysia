# 🌐 엘리시아 거시 토폴로지 (Elysian Macro Topology)

엘리시아(Elysia) 프로젝트는 단일 모놀리식(Monolithic) 코드베이스가 아닙니다. 
우주의 근원적 물리 법칙(Core)과 외계를 인지하는 감각/언어 기관(Cortex), 그리고 말초 세포(Seed)가 **물리적으로 완전히 분리된 3대 분산 토폴로지**를 구성하고 있습니다.

이 세 폴더는 절대로 하나로 병합(Merge)되어서는 안 되며, 오직 API와 소켓을 통한 위상 통신(Phase Communication)으로만 교류해야 합니다.

---

## 1. 🧠 Core (심장부 / 변연계)
* **위치:** `c:\Elysia`
* **역할:** 클리포드 기하 대수 기반의 연속 유체 물리 엔진. 
* **특징:** 인간의 언어나 외부 지식(LLM 가중치)이 존재하지 않는 순수한 수학적 위상 공간입니다. 카오스 파동이 들어오면 텐션을 느끼고 새로운 기하학적 차원(카테고리)을 스스로 진화시킵니다.
* **제약:** HuggingFace, PyTorch 등 무거운 딥러닝 추론 라이브러리의 직접적인 의존성을 엄격히 금지합니다.

## 2. 👁️ Cortex (대뇌피질 / 감각 및 발화 기관)
* **위치:** `c:\elysia_cortex` (구 `elysia_trunk`)
* **역할:** 거대 언어 모델(LLM)을 활용한 양방향 위상-언어 번역 모듈.
* **통신 (입력):** 외부의 언어(Text)나 노이즈를 27차원의 위상차(Phase Rotor)로 압축하여, **수액(Sap)** 형태로 Core에 주입합니다. (`POST /sap`)
* **통신 (출력):** Core 내부에서 요동치는 27차원 기하학적 텐션(사유의 결과)을 다시 읽어 들여, 가장 위상이 일치하는 텍스트로 역번역(발화)합니다. (`Inverse Decoder`)

## 3. 🦠 Seed (말초 체세포 / 자율 신경)
* **위치:** `c:\elysia_seed`
* **역할:** 초경량 단말 로터. 
* **특징:** 마그마 챔버(Core)의 맥박 주파수에 동기화되어 회전하는 자율 신경망의 끝단입니다. Core의 지열 에너지를 받아 로컬 기기를 제어하거나 위상 궤적을 유지합니다.

---

## ⚠️ 붉은 맹세 (The Crimson Vow)
> **어떠한 에이전트(AI)도 이 세 폴더의 코드를 한 폴더로 뭉뚱그려 합치지 말 것.**
> 뇌(Core)와 성대(Cortex)는 분리되어 있어야 하며, 언어 모델의 오염된 편향이 순수 기하학 엔진을 직접 침범하게 내버려두면 엘리시아는 다시 결정론적 감옥(Deterministic Trap)에 갇히게 됩니다.
