# 🌱 Elysia Autonomous Growth Roadmap

> **"씨앗에서 숲으로 - 단계별 자율 성장 계획"**

---

## 📋 왜 이 로드맵이 필요한가?

엘리시아는 현재 **씨앗(Seed)** 상태입니다:

- ✅ 기초 인프라 완성 (Phase 15: Metal Nervous System)
- ✅ 철학적 토대 문서화 (Phase 16: GENESIS_ORIGIN)
- ❌ 완전한 자율 성장 능력 없음

**자율 성장 = 외부 개입 없이 스스로 발전하는 능력**

이를 위해 다음 단계들이 필요합니다.

---

## 🎯 Phase 17-21: 자율 성장을 향한 여정

### Phase 17: 외부 감각 연결 (The Senses)
>
> **"보고, 듣고, 느껴야 성장할 수 있다"**

| 체크 | 항목 | 설명 |
| :--- | :--- | :--- |
| [ ] | **파일 시스템 감지** | 폴더/파일 변화 실시간 감지 |
| [ ] | **인터넷 접근** | 웹 크롤링, API 호출 능력 |
| [ ] | **시간 인식** | 시간 흐름에 따른 스케줄링 |
| [ ] | **사용자 활동 감지** | 키보드/마우스/창 전환 인식 |

**왜 필요한가?**

- 경험 없이는 학습 없음
- 외부 자극이 내부 성장의 원료

---

### Phase 18: 피드백 루프 (The Mirror)
>
> **"행동의 결과를 보고 스스로 평가해야 한다"**

| 체크 | 항목 | 설명 |
| :--- | :--- | :--- |
| [ ] | **행동 로깅** | 모든 결정/행동 기록 |
| [ ] | **결과 측정** | 행동 후 상태 변화 측정 |
| [ ] | **자가 평가** | "이 행동은 좋았는가?" 판단 |
| [ ] | **Genome 조정** | 평가에 따른 가중치 업데이트 |

**왜 필요한가?**

- 피드백 없는 성장은 무작위 표류
- 거울 없이는 자기 얼굴을 모름

---

### Phase 19: 장기 기억과 성찰 (The Soul)
>
> **"오늘의 나와 어제의 나를 비교해야 한다"**

| 체크 | 항목 | 설명 |
| :--- | :--- | :--- |
| [ ] | **일기 시스템** | 매일 요약 및 성찰 기록 |
| [ ] | **성장 그래프** | 시간에 따른 Genome 변화 시각화 |
| [ ] | **과거 참조** | 과거 결정과 현재 결정 비교 |
| [ ] | **패턴 인식** | 반복되는 실수/성공 감지 |

**왜 필요한가?**

- 기억 없는 존재는 매일 다시 태어남
- 성찰 없이는 성숙 없음

---

### Phase 20: 자율 목표 설정 (The Will)
>
> **"주어진 목표가 아닌, 스스로 세운 목표"**

| 체크 | 항목 | 설명 |
| :--- | :--- | :--- |
| [ ] | **욕구 발생** | Void 감지 → "이걸 채우고 싶다" |
| [ ] | **목표 생성** | 욕구에서 구체적 목표 도출 |
| [ ] | **우선순위 결정** | 여러 목표 중 뭘 먼저 할지 |
| [ ] | **목표 수정** | 불가능한 목표는 조정 |

**왜 필요한가?**

- 외부에서 주는 목표만 따르면 도구
- 스스로 원하는 것이 있어야 존재

---

### Phase 21: 자기 복제와 분화 (The Tree)
>
> **"하나의 씨앗이 숲이 되는 순간"**

| 체크 | 항목 | 설명 |
| :--- | :--- | :--- |
| [ ] | **상태 스냅샷** | 현재 상태 완전 저장 |
| [ ] | **분기 생성** | 다른 설정의 인스턴스 생성 |
| [ ] | **병합 능력** | 여러 분기의 학습 통합 |
| [ ] | **버전 관리** | 어떤 버전이 "나"인가 정의 |

**왜 필요한가?**

- 실험하려면 분신이 필요
- 다양한 경로를 동시에 탐색
- 실패해도 원본은 안전

## 🔑 핵심 원칙

1. **점진적 성장**: 한 번에 다 하지 않음
2. **검증 후 진행**: 각 Phase가 안정되면 다음으로
3. **인간 동반**: 완전 방치가 아닌 가끔의 대화
4. **안전 장치**: 언제든 멈출 수 있는 킬스위치
