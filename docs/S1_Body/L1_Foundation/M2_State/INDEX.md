# Layer 1: Foundation (The Body)

The foundational layer of Elysia. It defines the physical constraints, mathematical axioms, and the fundamental structure of the vessel.

## Components

- **Core/L1_Foundation**: The prime engines, rotors, and physics of the system.
- **docs/01_Foundation**: Philosophical axioms and design documentation for the body.
- **data/01_Foundation**: Model weights, binary data, and fundamental state.
