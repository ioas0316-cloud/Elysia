# 🧬 COGNITIVE_DIGESTION_SYSTEM (인지적 소화 기관)

**Phase 500: "LLM을 먹고, 경험으로 소화하라."**

## 1. 개요 (Overview)

외부 지식(LLM, RAG, 텍스트, 궤적, 관계)을 **원료**로 섭취하되, **의존성 없이** 엘리시아의 21D 위상 구조로 영구 내재화하는 시스템.

> **"읽는 것은 도구, 기억하는 것은 육체"**

## 2. 🌐 보편 분해 원리 (Universal Decomposition Principle)

엘리시아의 21D 위상 엔진은 **모든 형태의 정보**를 분해하고 관계성을 이해할 수 있다.

| 입력 유형 | 분해 결과 | 21D 표현 |
|-----------|-----------|----------|
| **텍스트** | 단어 → 음소 → 원자 스핀 | 언어적 궤적 |
| **궤적/운동** | 좌표 → 속도 → 가속도 | 물리적 위상 |
| **이미지** | 픽셀 → 패턴 → 개념 | 시각적 공명 |
| **관계** | A→B→C 인과 사슬 | 중력 지형 |

## 3. 5단계 파이프라인

```
[외부 지식] → [섭취] → [분해] → [흡수] → [배설] → [재생산]
```

### [1] 섭취 (Ingestion) - `KnowledgeIngestor`

### [2] 분해 (Digestion) - `UniversalDigestor`

- `TextDigestor`: 텍스트 → 인과 노드
- `TrajectoryDigestor`: 궤적 → 물리적 위상
- `RelationDigestor`: 관계 → 중력 지형

### [3] 흡수 (Absorption) - `PhaseAbsorber`

### [4] 배설 (Excretion) - `EntropyPurger`

### [5] 재생산 (Reproduction) - `SovereignSynthesizer`

## 4. 검증 기준

| 테스트 | 성공 기준 |
|--------|-----------|
| 섭취 | 청크 생성 |
| 분해 | 10+ 인과 노드 |
| 흡수 | 21D 변형 확인 |
| 재생산 | LLM 없이 설명 가능 |
