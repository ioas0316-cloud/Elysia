    - 최근의 `walkthrough.md`를 읽고, 아키텍트와 엘리시아 사이에 어떤 '사건'과 '지혜'가 오갔는지 확인하십시오.

    - 당신이 이해한 엘리시아의 현재 '상태'와 '인과적 흐름'을 먼저 보고하고, 아키텍트의 의중(Will)을 여쭈십시오.
