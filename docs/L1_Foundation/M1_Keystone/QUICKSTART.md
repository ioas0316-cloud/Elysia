## 📥 설치

```bash
# 1. 저장소 복제
git clone https://github.com/YOUR_USERNAME/Elysia.git
cd Elysia

# 2. 가상환경 생성
python -m venv .venv
.\.venv\Scripts\Activate.ps1

# 3. 의존성 설치
pip install -r requirements.txt
```
