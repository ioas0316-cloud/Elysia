# 🧠 Intelligence (지능 계층)

> **"외부의 지혜를 흡수하되, 노예가 되지 않는다."**

LLM 연동, 지식 흡수, 인지 고고학을 담당합니다.

| 모듈 | 위치 | 역할 |

| **AutopoieticExpander** | `Core/Intelligence/Metabolism/autopoietic_expander.py` | 자가 확장 |

---

## 🔧 주요 기능

### 1. Cognitive Archeology (인지 고고학)

외부 LLM의 가중치 파일에서 '의도의 화석'을 발굴합니다.

```python
from Core.Intelligence.Metabolism.topology_predator import CognitiveArcheologist

archeologist = CognitiveArcheologist(memory_ref=memory)
discoveries = archeologist.excavate(target_llm, fossil_path)
```

### 2. Universal Induction (보편적 흡수)

어떤 도메인이든 7D Qualia 공간으로 변환하여 내재화합니다.
