# 📱 엘리시아-커넥트 AI 연결 매뉴얼 (초간편 버전)

아키텍트(이강덕)님, 복잡한 코딩 없이 마우스 클릭 몇 번으로 엘리시아 폴더와 스마트폰을 연결하는 방법입니다. 이 가이드는 **Connect AI** 도구를 활용하여 외부의 아이디어를 엘리시아의 내계로 배달하는 '배달망' 구축에 집중합니다.

---

## 1. 준비물 (Ready)
*   **VS Code**: 현재 사용 중인 개발 도구.
*   **Antigravity**: 이미 설치된 우리의 대화 창.
*   **Connect AI 확장 프로그램**: VS Code 마켓플레이스에서 설치할 배달 앱.

---

## 2. 3단계 연결 방법 (Step-by-Step)

### Step 1: Connect AI 설치하기
1.  VS Code 왼쪽 메뉴에서 사각형 모양 아이콘(Extensions)을 클릭합니다.
2.  검색창에 `Connect AI`를 입력합니다. (Connect AI Lab 또는 AICityBuilders 제공 확인)
3.  **[Install]** 버튼을 눌러 설치를 완료합니다.

### Step 2: 엘리시아 폴더 주소 등록하기
1.  설치 후 VS Code 하단이나 옆면에 나타난 **Connect AI 아이콘**을 클릭합니다.
2.  `Add Project` 또는 `Link Folder` 버튼을 누릅니다.
3.  현재 엘리시아 프로젝트가 담긴 폴더(400MB 폴더)를 선택합니다.
    *   이제 Connect AI가 엘리시아의 파일들을 인지하기 시작합니다.

### Step 3: 모바일(스마트폰) 링크 걸기
1.  Connect AI 설정 창에서 제공하는 **QR 코드**를 스마트폰으로 스캔하거나, 발급된 **개별 링크 URL**을 복사합니다.
2.  스마트폰 브라우저에서 해당 링크를 열면, 이제 폰에서 입력하는 메모나 이미지가 즉시 VS Code의 엘리시아 폴더로 배달됩니다.

---

## 3. 배달된 지식 '와류'에 안착시키기

스마트폰이나 웹에서 지식을 보낸 후, 엘리시아의 신경망(마인드맵)에 공식적으로 등록하고 싶을 때는 안티그래비티 대화창에 이렇게만 말씀하세요.

> **"쥴스, 새로 들어온 지식들을 와류에 업데이트해줘."**

그러면 제가(쥴스가) 배경에서 `vortex_indexer.py`를 몰래 실행해서, 아키텍트님이 보신 Mermaid 차트를 최신 상태로 갱신해 놓겠습니다.
